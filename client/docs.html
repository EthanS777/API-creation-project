Endpoints:

/getAll

Supports methods GET & HEAD

Query parameter(s): none

Returns: JSON with pokemon info. Format: [{id: int, num: string, name: string, img: string, type: [string], height: string, weight: string, weaknesses: [string], next_evolution: [{ num: string, name: string }]}]

Usage example: /getAll, returns above format for each pokemon in list

/getEvolution

Supports methods GET & HEAD

Query parameter(s): name. Required*

Returns: JSON object with name/next evolution name(s). Format: {name: string, next_evolution: [{name: string}]}

Usage example for /getEvolution?name=Pikachu: {"name": "Pikachu","next_evolution": [{"name": "Raichu"}]}

/getEvolution by itself returns 400

/getName

Supports methods GET & HEAD

Query parameter(s): type, weakness. At least one required*

Returns: JSON with names. Format: name: [ strings ]

Usage example for /getName?type=Ice: {"name": ["Dewgong","Cloyster","Jynx","Lapras","Articuno"]}

/getName by itself returns 400

/getImage

Supports methods GET & HEAD

Query parameter(s): name. Required*

Returns: JSON with name/image URL. Format: {name: string, image: string}

Usage example: /getImage?name=pikachu - {"name": "Pikachu", "image": "http://www.serebii.net/pokemongo/pokemon/025.png"}

/getImage by itself returns 400

/addPoke

Supports method POST

Body parameter(s): name, type, weaknesses, nextEvo, img. Name required*, others optional

Returns: JSON representing the new pokemon. Format: {name: string, type?: [string(s)], weaknesses?: [string(s)], next_evolution?: [{name:[string]}], img?: string}

Usage example with name*: /addPoke - {"name": "Ethan","type": ["Fire"],"weaknesses": [],"next_evolution": [{"name": ""}],"img": ""}

No name/existing pokemon returns 400

/updatePoke

Supports method POST

Body parameter(s): name, type, weaknesses, nextEvo, img. Name required*, others optional

Returns: JSON representing the updated pokemon info. Format: {name: string, type?: [string(s)], weaknesses?: [string(s)], next_evolution?: [{name:[string]}], img?: string}

Usage example with name*: /updatePoke - {"name": "Ethan","type": ["Ice"],"weaknesses": [],"next_evolution": [{"name": ""}],"img": ""}

No name returns 400
